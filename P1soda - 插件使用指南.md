## 主机信息收集插件

目前只支持Windows基本信息收集

需要的参数如下

```
-plg infospy //指定插件
-collect basic // 基础信息收集模式
```

基础信息收集包括

```
系统基本信息、磁盘和共享信息、网络信息（网卡、arp缓存等等）、补丁、环境变量信息
```

命令行使用示例，`-plg infospy -collect basic`，

报告以`Html`形式输出在当前目录，命名格式为 `Hostnmae_Report.html`
